# bayesToolkit

## Bayesian Probability Computation and Visualization Tools for R

bayesToolkit provides a streamlined interface for computing Bayes’
Theorem, generating contingency tables, and producing clear
visualizations that support analytical workflows, decision modeling, and
probabilistic reasoning. The package is lightweight, dependency‑free,
and designed for clarity, reproducibility, and operational use.

## Key Capabilities

- Compute posterior probabilities using Bayes’ Theorem
- Generate 2×2 contingency tables from prevalence, sensitivity, and
  false‑positive rates
- Produce high‑quality visualizations:
  - Grouped bar charts
  - Heatmaps
  - Posterior probability plots
- Simple, transparent API suitable for analytics, modeling, and
  reporting

## Installation

This package is stored inside a subdirectory of a larger GitHub
repository. Install it using `devtools::install_github()` with the
`subdir` argument.

``` r

install.packages("devtools")    # required if not already on system

devtools::install_github(       # installation code
  "mcvanderbilt/code-library",
  subdir = "r/bayesToolkit"
)

library(bayesToolkit)           # load the package
```

## Example Usage

``` r

result <- bayes(
  prevalence     = 0.07,
  sensitivity    = 0.92,
  false_positive = 0.07
)

# Posterior probability
result$posterior_probability

# Contingency table
result$contingency_table

# Visualizations
result$barchart
result$heatmap
result$posterior_plot
```
